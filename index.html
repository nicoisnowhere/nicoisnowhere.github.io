Here are some fun facts:

  • Binomial nomenclature involves a two-term name for classifying a species.

  • These names are commonly referred to as binomial names, scientific names, or Latin names.

  • The scientific name for us is Homo sapiens.

  • Scientific names are written with the first word capitalized and the second in lower case.

  • If you can, make sure to write it in Italics (Homo sapiens instead of Homo sapiens).

  • Scientific names can be shortened with initials, but spell out the last word for clarity: H. sapiens.
